# GoTop

一个带有自定义菜单的滚动到顶部按钮

A scroll-to-top button with a custom menu

### [安装 / install](https://raw.githubusercontent.com/REDDRAGON-HL/GoTop/main/GoTop.user.js)

## 使用 / Usage

- **左键点击**：滚动到顶部
- **右键点击**：打开设置菜单
- **打开菜单后，拖动按钮**：调整位置
- 设置菜单里可打开「滚到底部」按钮、编辑排除站点列表、自定义 CSS 样式

- **Left-click**: scroll to top
- **Right-click**: open the settings menu
- **Drag the button while the menu is open**: reposition it
- The menu lets you toggle a "scroll to bottom" button, edit the exclude list, and customize CSS
